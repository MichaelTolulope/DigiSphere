## üé® Customization

### Colors
The template uses a blue and purple gradient theme. To change colors:

1. **CSS Variables**: Update the color values in `assets/css/style.css`
2. **Tailwind Classes**: Modify color classes in HTML files
3. **Gradient**: Update the `.hero-gradient` class for different background colors

### Content
1. **Text Content**: Edit the HTML files to update headings, descriptions, and copy
2. **Images**: Replace placeholder content with your own images in `assets/img/`
3. **Logo**: Update the "SaaSify" text in the navigation with your brand name
4. **Contact Information**: Update email addresses and phone numbers in contact sections

### Styling
1. **Fonts**: Change Google Fonts import in the `<head>` section
2. **Spacing**: Modify Tailwind spacing classes (p-, m-, py-, px-, etc.)
3. **Components**: Customize button styles, cards, and form elements
